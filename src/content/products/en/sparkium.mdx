## Who is Sparkium for?
Sparkium is for anyone working with eLearning content. Whether you're creating, managing, or maintaining it.

- Educators & Trainers 
- Content Authors & Publishers
- LMS Administrators 
- Instructional Designers & Developers

If you work with eLearning content, Sparkium is built for you.


## Features

### Markdown-based
Content is written in a simple, text-friendly format that's easy to read and edit - think of it as writing in a plain document while keeping structure and formatting.
### Change-managed
Every update is tracked, so you can see what changed and easily manage revisions over time. No need to worry about losing previous versions!
### Structured
Organize your course in a clear, logical file system, making it simple to find, update, and maintain content efficiently.
### Portable
Courses aren't locked to the LMS. With Sparkium, you can export, modify, and update your content locally, then re-upload it to the LMS to keep your courses flexible and easy to maintain.

## Why Sparkium?

| Comparison        | SCORM                     | xAPI                        | Sparkium                       |
|---------------------------|---------------------------|-----------------------------|--------------------------------|
| **Ease of Content Creation** | 🟡 Requires LMS setup | 🟡 More flexible but needs technical setup | 🟢 Create in text-based format with clear structure |
| **Update & Change Management** | 🔴 Hard to update existing courses | 🟡 Can track some changes | 🟢 Content can be updated locally and re-uploaded seamlessly |
| **Structured Organization**     | 🔴 File structure not standardized       | 🟡 Partially structured | 🟢  Clear, logical file system makes content easy to find, update, and maintain |
| **Content Portability**     | 🔴 Tied to LMS         | 🟡 Can move between xAPI-enabled platforms | 🟢  Export, modify, and re-upload within supported LMS |
| **Technical Barrier**       | 🔴 Technical skills required | 🔴 Technical skills required | 🟢 Simple text-based workflow, user-friendly |
